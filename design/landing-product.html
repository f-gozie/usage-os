Privacy

Nothing leaves this machine.

It reads your window titles and the current browser tab, and writes them to a database on your Mac. There is no network in the data path at all — and it’s open source, so you can read the code and confirm it.

It never uses Screen Recording. It skips incognito and private windows, ignores anything you mark private, and auto-excludes password managers and banking. No cloud, no account, no telemetry. Ever.
