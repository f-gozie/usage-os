## 4. Release-flow notes for next time (v0.1.2+)
1. Bump `tauri.conf.json` + `package.json` + `src-tauri/Cargo.toml` (+ `cargo update -p usage-os`
   for the lockfile) + CHANGELOG; commit to main; tag `v<version>` on that commit.
2. `PATH="/usr/bin:$PATH" ./scripts/release-macos.sh` (sources `~/.appstoreconnect/usageos-signing.env`).
3. **Check the DMG staple** — if "dmg not stapled": `xcrun notarytool submit <dmg> --key
   $APPLE_API_KEY_PATH --key-id $APPLE_API_KEY --issuer $APPLE_API_ISSUER --wait` then
   `xcrun stapler staple <dmg>`.
4. Rename the DMG to dashes (`UsageOS-<v>-universal.dmg` — tauri emits underscores), upload the
   three assets, `gh release create v<v> --title ... --notes ...`.
5. **Landing hardcodes the versioned DMG URL** (`landing/src/pages/index.astro:15`) — it breaks the
   moment the new release becomes "latest"; update + push immediately after publishing.
   (Worth switching to a version-stable DMG asset name someday.)
6. Cask: version + sha256 via `gh api -X PUT repos/f-gozie/homebrew-tap/contents/Casks/usageos.rb`.
