# Impl-plan — Week view (M2, first Phase-2 build)

_Branch `phase1/bauhaus-day` (stacked on PR #13). Backend-first, mirroring the Day dial (D36). Design source of truth: `design/week.html`._

## Goal
A Week tab: a **week summary** (Active this week / Avg per day / Deepest day) + a 7-cell grid of **mini-dials** (one per day, Sun→Sat), today highlighted, click-a-day → opens it in the Day view. Reuses the dial geometry + the `rollup` engine — numbers in Rust (hard rule 6), the frontend only renders.

## Backend (Rust)
- **`rollup.rs`** — new pure functions + types:
  - `DaySlice { day_start: i64, active_secs: i64, deep_secs: i64, runs: Vec<ContextRun> }` — one day's mini-dial arcs + the two totals the summary needs. (`deep_secs` so "deepest day" is a Rust number.)
  - `WeekView { days: Vec<DaySlice>, total_active_secs, avg_active_secs, deepest_day: Option<i64> }` (`deepest_day` = index into `days`, or `None`).
  - `build_day_slice(day_start, events, contexts, projects) -> DaySlice` — reuses `build_runs`; sums active + deep (`DEEP_SLUG`).
  - `build_week_view(days: Vec<DaySlice>) -> WeekView` — total = Σ active; avg = total / 7; deepest = argmax `deep_secs` (>0).
  - Tests: day-slice totals/runs; week totals/avg; deepest argmax + `None` when no deep work.
- **`lib.rs`** — `get_week(db, day_starts: Vec<i64>, week_end: i64) -> Result<rollup::WeekView>`: frontend passes the 7 local midnights (DST-correct) + the exclusive week end; the command fetches each day's events (`get_activity_logs`, like `get_day`) and assembles. Register in `collect_commands!`. Regenerates `bindings.ts`.

## Frontend (React)
- **`lib/dates.ts`** — `startOfWeek(date)` (Sunday-based, matching the mockup) + `weekDays(date): Date[]` (7). App derives `day_starts`/`week_end` via `dayBounds`.
- **`lib/tauri.ts`** — re-export `get_week`, `WeekView`, `DaySlice`.
- **`hooks/useWeekData.ts`** — mirror `useDayData` (load + 30s refresh on today).
- **`components/dial/MiniDial.tsx`** — small 24h dial (80-viewBox, r≈29, the same `arcPath`/`polar`/`minutesSinceMidnight` from `@/lib/geometry`): idle track, run arcs w/ casing (R77), now-triangle on today. No centre figure (label sits below). Story + RTL test.
- **`views/WeekView.tsx`** — summary `StatTile`s (Active / Avg / Deepest day, Deepest in deep-blue) + the `wgrid` of day cells (MiniDial + "Sun 15" + active duration), today highlighted, click → `onOpenDay(date)`. Loading/empty/degraded states like `DayView`. RTL test.
- **`App.tsx`** — replace `<Placeholder title="Week" />` with `<WeekView date onOpenDay={(d) => { setDate(d); setView("day"); }} />`; header date "Week of N–N Month".
