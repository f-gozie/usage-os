# Impl-plan — M4 Settings (as built)

_Branch `phase2/m4-settings` (off `main` after squash-merging PR #13). Decision: D42._

## Scope
The four **buildable** Settings groups (no new native / Phase-3 plumbing): **Contexts & rules**,
**Privacy & exclusions**, **Appearance**, **Your data**. Deferred: **Tracking** (background-pause,
always-on idle gate D39, day-start D14, Accessibility/Automation TCC status) and **Local
intelligence** (Phase-3 sidecar).

## Backend (`src-tauri/src/`)
- **`db.rs`** — `Category`→`Context` (+`slug: Option<String>`, selected via `get_contexts`);
  `create_context`/`update_context` (new)/`delete_context`; `Rule.category_id`→`context_id`
  (SQL column unchanged); `find_category`/`reprocess_logs` read `rule.context_id`. New data-ownership
  fns: `export_events_csv<W: Write>` (streams rows; reuses `ACTIVITY_LOG_COLUMNS`+`row_to_activity_log`;
  `csv_field`/`csv_opt` RFC-4180 escaping; `io_to_db_err`), `delete_all_data(&mut Connection)`
  (one transaction: wipe `activity_logs`/`project_aliases`/`projects`/`sites`; preserve config).
- **`lib.rs`** — commands renamed/added: `get_contexts`/`create_context`/`update_context`/
  `delete_context`, `create_rule(context_id,…)`, `get/create/delete_exclusion`, `set_retention_days`
  (persist + `cleanup_old_data`, returns rows-deleted), `get_database_path(AppHandle)`,
  `export_events_csv(AppHandle, db)` (writes `usageos-export-<ts>.csv` next to the DB, returns path),
  `delete_all_data`. All added to `collect_commands![]`. `ActivityLog.category_id` left as-is (hot path).
- **`capabilities/default.json`** — `+ opener:allow-reveal-item-in-dir`.
- **`bindings.ts`** — regenerated (`cargo test export_bindings`).

## Frontend (`src/`)
- **Primitives** (`components/ui/`, 1:1 from `design/components.html`, each + story):
  `TextInput`, `Select` (custom dropdown, outside-click/Esc), `RadioGroup`, `Swatches`,
  `Modal` (+ danger header, Esc/overlay close), `Toggle` (built for the future Tracking group).
- **Settings atoms** (`components/settings/primitives.tsx`): `SettingGroup`/`SettingRow`/`Pill`/
  `IconButton`/`Tag`/`ModePill`/`AddRow`.
- **Modals** (`components/settings/`): `ContextEditorModal` (add/edit; colour hidden for canonical;
  rules diff → create/delete → `reprocessLogs`; delete only for user contexts), `ExclusionModal`
  (App/Site/Title × Exclude/Private), `DeleteAllModal` (type-`DELETE` gate).
- **`hooks/useSettingsData.ts`** — loads contexts+rules+exclusions+settings (`{data,loading,error,refresh}`).
- **`views/SettingsView.tsx`** — the four groups + notice region; reveal-DB/export via
  `@tauri-apps/plugin-opener` `revealItemInDir`; theme via `useTheme`. `lib/tauri.ts` wrappers added.
  `App.tsx` renders `<SettingsView/>` (deleted the dead `Placeholder`).
